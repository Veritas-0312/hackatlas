A structured guide to learning the 10 high-value vulnerability classes — prerequisites first, then targeted resources for each category.

Phase 01 — Foundations
Prerequisites Before Everything

Networking Fundamentals

HTTP/S request lifecycle, DNS, TCP/IP, headers, cookies, TLS. You must understand what happens when a browser talks to a server.

MDN — HTTP Overview Cloudflare Learning Center NetworkChuck (YouTube)

Web App Architecture

Frontend/backend separation, REST vs GraphQL APIs, sessions vs JWTs, CDNs, reverse proxies, microservices patterns.

roadmap.sh — Backend Roadmap Hussein Nasser — Web Architecture

Scripting (Python / JS)

Write HTTP clients, parse JSON, automate requests, build PoC scripts. You don't need to be a developer but you need working fluency.

Python Requests Docs learnpython.org javascript.info

Burp Suite Proficiency

Proxy, Repeater, Intruder, Decoder, Scanner. This is your primary tool. Invest serious time here before hunting anything.

PortSwigger — Getting Started TCM Security (YouTube)

OWASP Top 10 Baseline

Understand the classic categories conceptually. Not deep enough to hunt, but essential context for everything else you'll learn.

OWASP Top 10 (Official) PortSwigger Web Security Academy

Linux & CLI Basics

Shell scripting, curl, grep, pipes, process management. Most tools run on Linux and most targets expose Linux internals.

OverTheWire: Bandit The Linux Command Line (free book)

Recon & Enumeration

Subdomain enum, port scanning, tech fingerprinting, Shodan, OSINT. Understanding the attack surface comes before finding bugs.

subfinder — ProjectDiscovery Hacker101 — Free Coursework

Platform Basics (HackerOne / Bugcrowd)

Understand scope rules, disclosure policies, severity ratings (CVSS), and report writing before submitting anything.

HackerOne Hacker101 Bugcrowd Researcher Docs
